CSS stands for cascading style sheets. CSS allows the web creator to customize the layout, fonts, colors, and much more using CSS! Internal CSS is used to customize the layout of one webpage. External CSS is used to customize multiple webpages. Internal CSS is located inside of the head, whilst external CSS is located in its own separate CSS file.
